# v0.6.2.1 Release Notes

发布目标：MCP npm 分发 — 终端用户无需 clone 仓库即可 `npx` 使用；同步 CLI npm `0.5.1`。无 API / 小程序契约变更。

## 组件版本

| 组件                       | 版本       | 说明                                 |
| -------------------------- | ---------- | ------------------------------------ |
| Git `release_tag`          | `v0.6.2.1` | npm 分发发布火车                     |
| API                        | `0.3.0`    | 无变更                               |
| CLI                        | `0.5.1`    | npm `@xiaolinstar/ai-todo-cli@0.5.1` |
| 微信小程序                 | `0.7.0`    | 无本批变更（沿用 v0.7.0 上传）       |
| `@xiaolinstar/ai-todo-mcp` | `0.1.0`    | **npm 首发**；bin `ai-todo-mcp`      |
| `@ai-todo/agent-protocol`  | `0.4.0`    | 无变更                               |

## 主要变化

- 仓库包重命名：`@ai-todo/mcp` → **`@xiaolinstar/ai-todo-mcp`**（与 CLI 同 scope）。
- npm 发布物：`dist/index.js`（bundle）、`README.md`；`prepublishOnly` 构建 + smoke。
- `docs/mcp-setup.md`：终端用户默认 `npx -y @xiaolinstar/ai-todo-mcp`；维护者发布流程。
- `docs/agent-usage.md`：MCP 安装路径对齐 npm。

## 用户安装

```bash
npm install -g @xiaolinstar/ai-todo-cli
npx -y @xiaolinstar/ai-todo-mcp   # 或 npm i -g @xiaolinstar/ai-todo-mcp
```

配置见 [mcp-setup.md](../mcp-setup.md)。
