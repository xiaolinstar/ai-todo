# v0.5.0 Release Notes

发布目标：微信订阅消息提醒触达闭环 + 小程序分享。

## 组件版本

| 组件 | 版本 | 说明 |
|------|------|------|
| Git `release_tag` | `v0.5.0` | 本 tag |
| API | `0.2.4` | 订阅消息 worker、delivery 同步、部署自动启 worker |
| CLI | `0.4.0` | 无变更 |
| 微信小程序 | `0.5.0` | 提醒订阅、分享、改期重新授权、通知深链 |

## 变更

### API / Worker

- 对齐微信公共模板 **#15788**（`thing23` / `time2` / `thing13`）。
- 提醒 update / reschedule / complete / delete 时同步 notification delivery 队列。
- 到期时间格式化为微信 `time` 字段可接受的形式。
- CD：`AI_TODO_WECHAT_REMINDER_TEMPLATE_ID` 已配置时自动 `--profile notifications` 启动 worker。

### 微信小程序

- 创建提醒：订阅授权 + 写入 delivery。
- 编辑提醒改期：重新 `requestSubscribeMessage`。
- 订阅消息点击：`reminders?reminderId=` 深链至编辑页。
- 分享：Tab 页与「关于」支持发送给朋友与分享到朋友圈。

### 文档

- [v0.5.0-plan.md](./v0.5.0-plan.md)、[deploy.md](../deploy.md) 联调与验收说明。

## 非目标（下一批）

- **日程（Calendar）微信提醒**：与提醒共用同一订阅模板可行，实现方案另议（`v0.5.1` / `v0.6.0` 候选）。

## 生产配置

```bash
AI_TODO_WECHAT_REMINDER_TEMPLATE_ID=<公众平台模板 ID>
```

模板字段须与 worker 一致（见 deploy.md）。
