# v0.5.5 Release Notes

发布目标：小程序工程内聚 + v0.5.4 后续体验修复。

## 组件版本

| 组件 | 版本 | 说明 |
|------|------|------|
| Git `release_tag` | `v0.5.5` | 本 tag |
| API | `0.2.7` | 无变更 |
| CLI | `0.4.2` | 无变更 |
| 微信小程序 | `0.5.5` | 工具链收拢、运行时/构建警告修复 |

## 变更

### 微信小程序

- **工程内聚**：`miniprogram-ci`、构建/检查脚本、`sass`/`typescript` 依赖迁入 `apps/miniapp`；根目录保留 `pnpm check:wechat` 等快捷命令。
- **运行时**：`fetchReminders` 不再使用 `URLSearchParams`；左滑删除改用 `wx.getWindowInfo()`。
- **样式**：Sass `@import` 迁移为 `@use`，消除构建弃用警告。
- **CLI 预览/上传**：`apps/miniapp` 内 `pnpm preview` / `pnpm upload`（需 `ci.env` + 上传密钥）。

## 验证

- `pnpm check:wechat`
- 微信开发者工具提醒页无 console 报错
- （可选）`pnpm miniapp:preview` 扫码真机
