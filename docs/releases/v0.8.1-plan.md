# ai-todo v0.8.1 版本规划 — 设置一致性与账号体验

日期：2026-06-09  
状态：**候选**（建议在 **v0.8.0** 三态之后）

关联：[v0.7.0-v0.8.1-ux-iteration-plan.md](./v0.7.0-v0.8.1-ux-iteration-plan.md) § v0.8.1；[ux-issues-requirements-2026-06-09.md](../product/ux-issues-requirements-2026-06-09.md) §4、§7。

## P0

### 全局通知设置

- 设置页：**一个**「微信通知」区块，明确同时作用于提醒与日历。
- 创建/编辑提醒、日程：读取同一套全局默认；单项开关文案标明「覆盖全局」或 MVP 去掉分项混淆。
- 文档：`docs/api-design.md` 通知 settings 一节与 UI 对齐。

### 头像持久性

- 微信登录/资料刷新：**空 avatar 不覆盖**已有值。
- 用户主动清空才写空；列表/我的页 fallback 稳定。

---

## P1

- **隐私协议版本**：存储用户已同意版本；同版本重登不重复弹窗；版本升级再提示。
- **CLI 输出 polish**：reminder/calendar/contact/token 列表样式统一（v0.7.0 已英文化，本批对齐列宽/空态）。

---

## 非目标

- 新通知渠道、多语言、订阅模板变更
- 三态提醒（v0.8.0）
- 平台建联

- [ ] 重登不丢头像
