## 变更

- 小程序未登录/授权失败时展示明确登录提示，不再透传原始后端授权错误。
- 新建日程默认带结束时间，结束时间为开始时间后 1 小时。
- 日程创建和编辑时，开始时间变更会联动结束时间；用户手动修改结束时间后不再自动覆盖。
- 提醒和日历标题在列表中单行省略，创建/编辑标题输入限制为 80 字。
- 提醒列表展示来源角标，并按逾期 / 今天 / 未来分组。
- CLI 人类可读输出改为英文，并对列表长标题做截断；`--json` 输出结构不变。
- CLI 列表输出改为列式布局；提醒默认显示 `pending` 并按截止时间排序，`-a/--all` 显示所有状态。
- 提醒表格 ID 省略统一的 `rem_` 前缀，支持 4 位及以上短 ID 前缀定位，例如 `ai-todo reminder inspect c1f6`。
