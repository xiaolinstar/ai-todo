# ai-todo v0.1.3 发布说明

发布日期：2026-06-03
发布目标：微信小程序体验版 / 内测体验修订版

## 版本概述

`v0.1.3` 是 `v0.1.2` 后的内测体验修订版。本版本重点优化应用内联系人创建体验，将联系人创建从通讯录页底部弹层调整为独立页面，并同步补齐 CLI 联系人字段。

微信小程序发布版本号填写为 `0.1.3`，Git tag 使用 `v0.1.3`。

## 主要变更

- 新增独立“新建联系人”页面。
- 中央 `+ -> 添加联系人` 进入联系人创建页，通讯录页不再展示重复的新建按钮。
- 联系人创建表单按信息优先级拆分：
  - 基础联系方式：姓名、邮箱、电话；
  - 补充信息：公司、职位、备注。
- CLI 联系人创建 / 更新同步支持：
  - `--company`
  - `--job-title`
  - `--notes`
- 更新 Agent tool 描述和 `0.1.3` 规划文档。

## 隐私与合规

- 本版本仍只维护 ai-todo 应用内联系人。
- 不读取、不写入手机系统通讯录。
- 不新增手机号登录、定位、相册、摄像头、麦克风等隐私能力。

## 已知限制

- 提醒、日历和通讯录的信息再编辑能力暂未在小程序中实现。
- 再编辑能力规划进入 `0.1.4`。
