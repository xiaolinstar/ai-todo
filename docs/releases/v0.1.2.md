# ai-todo v0.1.2 发布说明

发布日期：2026-06-03
发布目标：微信小程序体验版 / 内测稳定打磨版

## 版本概述

`v0.1.2` 是 `v0.1.1` 后的内测稳定打磨版。这个版本重点收敛微信小程序隐私授权、个人资料设置、创建提醒/日历表单体验和服务端发布验收信息，让微信小程序客户端、FastAPI 服务端和 CLI 继续保持一致发布节奏。

微信小程序发布版本号填写为 `0.1.2`，Git tag 使用 `v0.1.2`。

## 主要变更

- 完成小程序隐私授权登录闭环：
  - 用户点击微信登录后，先完成隐私保护提示授权；
  - 用户拒绝时不调用 `wx.login`，也不创建登录态；
  - 用户同意后再继续微信登录和首次头像昵称确认。
- 新增正式的“个人资料”入口：
  - 从“我的”页进入；
  - 支持修改头像和昵称；
  - 头像昵称不再通过点击顶部资料直接编辑。
- 优化创建提醒和日历表单：
  - 备注 / 描述默认折叠；
  - 日期和时间合并为同一表单行；
  - 不提供显式剪切板读取入口，用户按微信原生习惯在输入框中自行粘贴。
- 优化发布验收信息：
  - 服务端 health 返回 `releaseTag` 和 `gitSha`；
  - 手动 CD 继续使用发布 tag 作为生产部署入口；
  - 文档补充 `0.1.2` 发布和隐私合规说明。

## 隐私与合规

- 当前版本不新增手机号、定位、相册、摄像头、麦克风、系统通讯录等能力。
- 当前版本不调用 `getClipboardData`；剪贴板文本仅来自用户在输入框中的主动粘贴。
- 小程序隐私保护指引已按当前数据使用范围整理：
  - 微信账号标识；
  - 微信头像、昵称；
  - 提醒；
  - 日历；
  - 应用内联系人；
  - CLI 访问令牌；
  - 订阅消息授权。

- 真实微信订阅消息投递仍不作为本版本发布门槛。

## 微信提审 / 体验版备注

ai-todo 是一款轻量的个人提醒、日历和通讯录小程序。当前版本用于小范围内测，用户可通过微信登录后同步个人提醒、日历和联系人数据；登录前会展示小程序隐私保护提示，首次登录后可确认头像和昵称作为小程序账户展示信息。本版本优化了个人资料设置入口、创建提醒和日历的表单体验，并继续保持小程序客户端与 FastAPI 服务端一致发布。
