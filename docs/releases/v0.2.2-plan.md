# ai-todo v0.2.2 版本规划（修订）

日期：2026-06-03  
原则：**轻量交付、复杂能力先写清产品设计再实现**。

## 背景

原草案包含「用户改时区 / 改 username / 按用户名添加联系人」。内测反馈：

1. 列表「今天」曾按手机本地日期算，与账户时区不一致，**体感未生效**（本批改为跟 API 账户时区，不提供改时区入口）。
2. **是否要让用户自助改时区、改 username** 尚未定稿（注册默认从哪来、改动的场景与风险）——**本批不做编辑**。
3. **按用户名搜人加联系人** 不宜做成微信加好友；既要能协作又要保护隐私，需单独产品方案，**本批不做**。

[CLI 令牌增强](./v0.3.0-plan.md) 仍后置；PAT **仅创建时显示一次**。

## P0（本批只做这些）

### 1. 账户时区：可改 + 全局一致

- `PATCH /v1/me/profile` 支持 `timezone`（IANA）。
- 小程序「我的 → 时区」：选择、保存；展示**该时区当前时刻**与「今天」日期。
- 提醒/日历/新建/编辑：日期与时刻**按账户时区**格式化与提交（非手机本地时区）。

### 2. 体验小修（若有内测清单项再追加）

- 从 0.2.1 验收反馈收集 1～2 条明确 bug/文案，单点修；不新开模块。
