# v0.8.6 Release Notes

发布日期：2026-06-27

## 组件版本

| 组件              | 版本     | 说明                                        |
| ----------------- | -------- | ------------------------------------------- |
| Git `release_tag` | `v0.8.6` | 微信提醒闭环                                |
| API               | `0.4.2`  | 新增提醒/日程微信提醒请求状态字段与状态汇总 |
| CLI               | `0.6.1`  | 无变更                                      |
| 微信小程序        | `0.8.6`  | 提醒/日程微信提醒开关、列表状态与授权入口   |

## 变更

- 提醒和日程摘要新增 `wechatNotifyRequested`、`wechatNotifyStatus`，列表和详情都能返回微信提醒状态。
- 小程序创建页、编辑页支持微信提醒开关；显式关闭时不会被“小程序默认开启”覆盖。
- 提醒列表、日历列表展示“待开启提醒 / 微信提醒已开启”，并支持补授权。
- 授权失败或用户取消时不阻塞保存，使用 toast 给出结果反馈。
