详见 [v0.4.0-plan.md](./v0.4.0-plan.md)。

## 面向用户

### 安装

```bash
npm install -g @xiaolinstar/ai-todo-cli
```

### 配置（一次）

在小程序 **我的 → CLI / Agent 访问令牌** 创建 PAT，将复制的 JSON 写入 `~/.ai-todo/settings.json`：

```json
{
  "url": "https://wodi.games",
  "token": "aitodo_xxx"
}
```

### 使用

```bash
ai-todo today
ai-todo whoami
ai-todo reminder list --json
```

## 变更摘要

- CLI 单包 esbuild 打包，发布 `@xiaolinstar/ai-todo-cli` 到 npm（全局命令仍为 `ai-todo`）
- `ai-todo help` 与文档以 `settings.json` 为唯一公开认证路径
- 小程序创建令牌后复制配置 JSON，不再推荐 `ai-todo login`
- `login` / `token` / `logout` 命令仍可在旧脚本中调用，但不再对外文档化

## 升级说明

- 已使用 `~/.ai-todo/settings.json` 的用户：安装 npm 包后直接可用，无需改配置
- 仍在用 `ai-todo login` 的用户：行为不变，建议改为直接维护 settings 文件
