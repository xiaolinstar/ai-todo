# ai-todo v0.1.4 发布说明

发布目标：微信小程序体验版 / 内测编辑能力修订版

## 版本概述

Git **`release_tag`：`v0.1.4`**（CD 与现状一致）。各组件 **L1 独立版本**如下：

| 组件                 | 版本    | 本批是否变更                                             |
| -------------------- | ------- | -------------------------------------------------------- |
| API (`ai-todo-api`)  | `0.1.3` | 否（契约已支持编辑/删除；health 新增 `apiVersion` 字段） |
| CLI (`@ai-todo/cli`) | `0.1.4` | 是                                                       |
| 微信小程序           | `0.1.4` | 是                                                       |

详见 [compatibility.md](./compatibility.md)。

`v0.1.4` 在 API `0.1.3` 基础上，由小程序与 CLI 补齐提醒、日历、联系人的再编辑与列表删除体验。

## 主要变更

- 小程序 `lib/api.ts` 增加 `fetch*` / `update*` 封装。
- 新增「编辑联系人」「编辑提醒」「编辑日程」页面。
- 通讯录列表点击进入编辑；提醒列表点击标题区域进入编辑；日历列表点击日程进入编辑。
- 已完成提醒可编辑标题、备注与联系人；截止时间编辑在已完成状态下隐藏。
- 日历、通讯录列表支持左滑删除，交互与提醒列表一致（滑动阈值、按压高亮、删除确认）。

## 隐私与合规

- 不新增隐私类目；不读取或写入系统通讯录/系统日历。

## CLI / Agent

- `ai-todo reminder update` 支持 `--title`、`--notes`、`--due`、`--remind`、`--contact`（与 API `PATCH /v1/reminders/{id}` 对齐）。
- `ai-todo reminder show` 输出截止、通知时间与关联联系人。
- 集成测试覆盖 `reminder show/update/delete` 与 `contact show/update`。
