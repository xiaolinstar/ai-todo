# ai-todo v0.3.0 发布说明（规划中）

发布目标：CLI 访问令牌生命周期 + API/CLI；微信小程序**可选**同期发布。

## 版本概述（计划）

| 组件 | 目标 L1 | 说明 |
|------|---------|------|
| Git `release_tag` | `v0.3.0` | |
| API | `0.2.0` | PAT 密文存储、空闲吊销、reveal 接口 |
| CLI | `0.3.0` | `ai-todo token` 子命令 |
| 微信小程序 | `0.3.0`（可选上传） | 令牌列表/详情/创建向导 |

详见 [v0.3.0-plan.md](./v0.3.0-plan.md)。

## 规划摘要

- 访问令牌支持**登录后随时查看**（服务端加密存储，非仅创建时一次）。
- 生命周期：创建、吊销、绝对到期、久未使用自动失效。
- 小程序：列表 + 详情 + 创建向导；弱化「只显示一次」阻断 UI。
