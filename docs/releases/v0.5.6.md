# v0.5.6 Release Notes

发布目标：miniprogram-ci 在 Node.js 25+ 下可正常编译，并简化上传密钥配置。

## 组件版本

| 组件              | 版本     | 说明                                      |
| ----------------- | -------- | ----------------------------------------- |
| Git `release_tag` | `v0.5.6` | 本 tag                                    |
| API               | `0.2.7`  | 无变更                                    |
| CLI               | `0.4.2`  | 无变更                                    |
| 微信小程序        | `0.5.6`  | miniprogram-ci Node 25 修复；上传密钥路径 |

## 变更

### 微信小程序

- **Node.js 25+**：新增 `scripts/run-ci.mjs`，通过 `--localstorage-file` 与 `NODE_OPTIONS` 让 miniprogram-ci fork 子进程继承 Web Storage 配置，修复 `localStorage.getItem is not a function`。
- **上传密钥**：支持将 `private.wx*.key` 放在 `apps/miniapp/` 并自动识别；相对路径 `./private.wx*.key` 解析；gitignore 与文档更新。

## 验证

- `pnpm check:wechat`
- Node 25 下 `pnpm miniapp:preview` 编译阶段无 localStorage 报错（上传仍受 IP 白名单约束）
