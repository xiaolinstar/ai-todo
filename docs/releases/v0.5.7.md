## 变更

### 合规

- 微信公众平台《用户隐私保护指引》（2026-06-07 文案）**审核通过**；真源见 [privacy-compliance.md](../privacy-compliance.md)。

### 微信小程序

- **custom-tab-bar**：组件样式改为 `@use todo-design-tokens`，避免 `page` 选择器 wxss 告警。
- **page.json**：移除无效的 `enableShareTimeline`（由页面 `onShareTimeline` 提供能力）。
- **分享**：`showShareMenu` 仅开启「分享给朋友」；体验/正式版在 App 启动时统一调用；develop 环境跳过以减少 DevTools timeout 噪声。
- **隐私回退**：`showModal` 失败时 resolve disagree，避免 privacy API pending。
- **工程**：默认调试基础库 `3.15.0`；README 说明 3.15.1/3.15.2 已知 timeout 日志。
