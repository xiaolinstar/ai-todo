# v0.4.4 Release Notes

发布目标：API 公共接入页与备案信息优化。

## 组件版本

| 组件 | 版本 | 说明 |
|------|------|------|
| Git `release_tag` | `v0.4.4` | 待 CI 通过后标记 |
| API | `0.2.3` | 公共首页、favicon、静态资源和备案展示优化 |
| CLI | `0.4.0` | 无变更 |
| 微信小程序 | `0.4.0` | 无变更 |

## 变更

- 将 FastAPI 根路径升级为响应式公共接入页。
- 增加 `/favicon.ico` 与 `/favicon.svg`，复用 API 静态网站 icon。
- 挂载 `/static` 并将 API 页面 icon、公安备案图标打入 Python wheel。
- 展示 ICP 备案与公安联网备案信息。
- 优化 CLI / Agent 三步接入说明：小程序创建 PAT、npm 安装 CLI、粘贴 PAT 并验证。

## 验证

- `pnpm test:api`
- `pnpm typecheck`
- `cd apps/api && .venv/bin/python -m ruff check src tests`
- Docker 本地服务 `http://localhost:3100/` 页面、favicon、静态图标与 `/v1/health` 验收通过。
