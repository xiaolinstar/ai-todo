# ai-todo v0.4.1 发布说明

Git 发布火车：npm 包名与文档对齐。各组件 **L1 版本不变**。

## 组件版本

| 组件 | 版本 | 本批是否 bump |
|------|------|---------------|
| Git `release_tag` | `v0.4.1` | — |
| API | `0.2.0` | 否 |
| CLI | `0.4.0` | 否 |
| 微信小程序 | `0.4.0` | 否（可选上传） |

## 变更摘要

- npm 发布包名确定为 **`@xiaolinstar/ai-todo-cli`**（无 scope 的 `ai-todo-cli` 已被占用；`@ai-todo` 组织未创建）
- 仓库内 workspace 名、CI filter、文档、help 与 npm 一致
- 全局命令仍为 **`ai-todo`**

## 用户安装

```bash
npm install -g @xiaolinstar/ai-todo-cli
```

配置方式不变：编辑 `~/.ai-todo/settings.json`。见 [v0.4.0.md](./v0.4.0.md)。

## 与 v0.4.0 的关系

| | v0.4.0 | v0.4.1 |
|---|--------|--------|
| Git tag | 首次 npm + settings 优先 UX | 包名/文档对齐 |
| CLI L1 | 0.4.0 | 0.4.0 |
| npm 包名（实际） | 用户侧已以 `@xiaolinstar/ai-todo-cli@0.4.0` 发布 | 仓库与 npm 命名一致 |

若需让 `ai-todo help` 与 npm 上已安装版本完全一致，可另发 CLI **L1 `0.4.1`**（optional patch），不强制与本 Git tag 同号。
