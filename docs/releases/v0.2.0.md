# ai-todo v0.2.0 发布说明

发布目标：微信小程序体验版 / 内测体验增强版

## 版本概述

| 组件              | 目标 L1         | 说明                 |
| ----------------- | --------------- | -------------------- |
| Git `release_tag` | `v0.2.0`        | CD 不变              |
| 微信小程序        | `0.2.0`         | 我的页与设置子页     |
| API               | `0.1.3`（预计） | 复用已有通知设置 API |
| CLI               | `0.1.4`（预计） | 无强制变更           |

详见 [v0.2.0-plan.md](./v0.2.0-plan.md)、[compatibility.md](./compatibility.md)。

## 主要变更

- 「我的」改为**设置枢纽**：分组菜单 + 二级页，底部**退出登录**；PAT 迁至「CLI / Agent 接入」。
- 资料卡展示昵称 + 平台用户名；个人资料页可查看/复制用户名。
- 设计令牌：`styles/tokens.scss`、`lib/design-tokens.ts`；禁止 SCSS/TS 硬编码色值（`check:wechat`）。
- 已实现二级页：通知设置、关于、Agent 令牌；开发选项（仅 develop）。
- 占位页（时区、提醒/日历/通讯录偏好、账号与安全）：`settings-placeholder`，按小版本迭代填实。
- 小程序 L1：**0.2.0**（微信上传版本号填写 `0.2.0`）。

## 已知限制

- 通知开关不替代微信订阅消息授权；创建提醒时仍需按流程订阅。
- 编辑提醒仍不触发重新订阅（同 0.1.4）。
