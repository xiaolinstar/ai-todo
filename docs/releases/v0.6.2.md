# v0.6.2 Release Notes

发布目标：stdio MCP Server — Cursor / Claude Desktop 等宿主**零 shell** 调用 ai-todo；无 API / 小程序契约变更。

## 组件版本

| 组件                       | 版本     | 说明                                         |
| -------------------------- | -------- | -------------------------------------------- |
| Git `release_tag`          | `v0.6.2` | MCP 接入面发布火车                           |
| API                        | `0.3.0`  | 无变更                                       |
| CLI                        | `0.5.0`  | 无变更（MCP 子进程调用）                     |
| 微信小程序                 | `0.6.0`  | 无需重传                                     |
| `@ai-todo/agent-protocol`  | `0.4.0`  | 无变更                                       |
| `@xiaolinstar/ai-todo-mcp` | `0.1.0`  | 新包，bin `ai-todo-mcp`（v0.6.2.1 npm 首发） |

## 主要变化

- 新包 `packages/mcp`：`@modelcontextprotocol/sdk` stdio Server，包装 `ai-todo --json`。
- P0 工具（12 个）：`whoami`、`today`、`reminder_find` / `reminder_create` / `reminder_create_sourced`、列表与按来源更新/完成、`contact_search`、`calendar_today` / `calendar_create`。
- 鉴权：`AI_TODO_TOKEN` / `~/.ai-todo/settings.json`，与 CLI 一致。
- `docs/mcp-setup.md`：Cursor / Claude Desktop 配置示例。
- `docs/agent-usage.md`、`skills/ai-todo/SKILL.md`：优先 MCP，无 MCP 时 shell。
- 根脚本 `pnpm mcp:build`、`pnpm mcp:smoke`；CI `scan-node` 增加 MCP 构建冒烟。

## 发布说明

- 本批**不要求** CD 或微信小程序上传（组件 L1 与 v0.6.1 相同）。
- 验收：`pnpm mcp:build`、`pnpm mcp:smoke`、`pnpm test:api`。

详见 [v0.6.2-plan.md](./v0.6.2-plan.md)。
