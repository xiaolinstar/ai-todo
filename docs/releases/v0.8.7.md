## 变更

- 新增 Reminder Tag 词表与 `tagNames[]` 写入能力，支持创建/更新提醒时整组同步标签。
- 新增提醒全文搜索与标签筛选：`q` 匹配标题、备注、标签、跟踪文本；`tag` 做精确标签筛选。
- 新增事项跟踪条目 `track-entries`，支持按用户时区生成 `MM-DD` 前缀的追加记录。
- CLI 对齐新增 `--tag` / `--q` / `reminder track add` 能力。
- 小程序新增搜索栏、标签 chip 筛选、标签输入与跟踪条目 UI。
- 修复头像缓存与展示稳定性问题，避免已设置头像在重新进入小程序后丢失或被空值覆盖。
- 修复提醒事项时间错位问题，确保提醒创建、编辑和列表展示按账户时区一致处理。
- 延续 v0.8.6 微信提醒闭环能力。
