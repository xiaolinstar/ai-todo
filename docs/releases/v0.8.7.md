# v0.8.7 Release Notes

发布日期：2026-06-29

## 组件版本

| 组件              | 版本     | 说明                     |
| ----------------- | -------- | ------------------------ |
| Git `release_tag` | `v0.8.7` | 小程序修复版             |
| API               | `0.4.2`  | 无变更                   |
| CLI               | `0.6.1`  | 无变更                   |
| 微信小程序        | `0.8.7`  | 头像持久化与提醒时间修复 |

## 变更

- 修复头像缓存与展示稳定性问题，避免已设置头像在重新进入小程序后丢失或被空值覆盖。
- 修复提醒事项时间错位问题，确保提醒创建、编辑和列表展示按账户时区一致处理。
- 延续 v0.8.6 微信提醒闭环能力；本次不引入 API/CLI 契约变更。
