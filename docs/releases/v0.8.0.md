# v0.8.0 Release Notes

发布目标：提醒三态工作流 — **未完成 / 处理中 / 已完成**，API、CLI、小程序、Agent/MCP 对齐。

## 组件版本

| 组件                       | 版本     | 说明                                        |
| -------------------------- | -------- | ------------------------------------------- |
| Git `release_tag`          | `v0.8.0` | 状态模型火车                                |
| API                        | `0.4.0`  | 新增 `in_progress`；`completed_at` 语义收紧 |
| CLI                        | `0.6.0`  | `--status in_progress`；英文状态标签        |
| 微信小程序                 | `0.8.0`  | 三段筛选 + 列表/编辑页状态切换              |
| `@ai-todo/agent-protocol`  | `0.5.0`  | status 工具与 guidelines                    |
| `@xiaolinstar/ai-todo-mcp` | `0.2.0`  | list/update status schema                   |

## 主要变化

- API：`status` 合法值含 `in_progress`；离开 `completed` 清空 `completed_at`。
- 小程序：提醒 Tab **未完成 / 处理中 / 已完成**（三 Tab 统一数量角标）；编辑页状态分段选择；创建提醒默认无截止时间。
- CLI：`reminder update --status in_progress`；列表显示 `Not started` / `In progress` / `Completed`。
- Agent：Skill、agent-protocol、MCP 对齐新 status。

## 发布说明

- **需要 CD**（API `0.4.0`）。
- **需要**微信小程序上传 `0.8.0`。
- npm：`@xiaolinstar/ai-todo-cli@0.6.0`、`@xiaolinstar/ai-todo-mcp@0.2.0`。

详见 [v0.8.0-plan.md](./v0.8.0-plan.md)。
