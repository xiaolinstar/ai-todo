# v0.5.3 Release Notes

发布目标：通讯录全量展示 + CLI 分页与 help 精简。

## 组件版本

| 组件 | 版本 | 说明 |
|------|------|------|
| Git `release_tag` | `v0.5.3` | 本 tag |
| API | `0.2.6` | 通讯录 REST 全量 + cursor |
| CLI | `0.4.2` | 分页输出、精简 help、移除 profile update、settings 迁移 wodi.games |
| 微信小程序 | `0.5.3` | 通讯录全量；通知记录标题与状态 |

## 变更

### API

- 通讯录列表支持全量返回与 cursor 分页。

### CLI

- `contact list` 等列表命令分页与摘要输出。
- 精简 help；移除 `profile update`；默认 API 域名迁移提示。

### 微信小程序

- 通讯录全量展示。
- 通知送达记录显示标题与中文状态。

## 验证

- `pnpm test:api`
- CLI `contact list` 分页与 REST 全量一致
