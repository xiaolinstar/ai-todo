## 变更

- 优化标签设置页的信息层次与视觉表现，增加概览、新建、搜索排序和列表状态区分。
- 优化标签详情页预览、名称编辑和颜色选择反馈。
- 优化提醒事项中的标签展示，仅显示少量标签并使用更轻量的胶囊样式，减少列表高度占用。
- 修复 API 标签使用统计：已软删除提醒不再计入标签使用数量。
