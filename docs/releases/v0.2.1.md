# ai-todo v0.2.1 发布说明

发布目标：微信小程序体验版

## 版本概述（计划）

| 组件 | 目标 L1 | 说明 |
|------|---------|------|
| Git `release_tag` | `v0.2.1` | CD 不变 |
| 微信小程序 | `0.2.1` | 使用偏好统一、通知分工、账号与安全 |
| API | `0.1.3` | 复用通知设置 API；无新路由 |
| CLI | `>=0.1.4` | 无强制变更 |

详见 [v0.2.1-plan.md](./v0.2.1-plan.md)。

## 计划变更摘要

- **使用偏好**：提醒、日历、通讯录合并为同一设置类（单页分区），不再三个占位菜单项。
- **通知设置**：仅保留渠道相关（微信总开关、模板/勿扰/送达摘要）；「新建默认通知」迁至使用偏好 · 提醒。
- **账号与安全**：由占位改为实装说明页。
- **提醒列表**：信息层次与逾期/元信息展示优化（P1）。

## 已知限制（预计继承）

- 时区、平台用户名设置 → `v0.2.2`。
- 编辑提醒不重新订阅；星标/撤销删除 → `v0.3.0`。
