# v0.6.1 Release Notes

发布目标：Agent 工具链闭环 — 对齐 v0.6.0 提醒来源能力，补齐工具目录、Skill 与文档；无 API / 小程序契约变更。

## 组件版本

| 组件 | 版本 | 说明 |
|------|------|------|
| Git `release_tag` | `v0.6.1` | Agent 接入面发布火车 |
| API | `0.3.0` | 无变更 |
| CLI | `0.5.0` | 无变更 |
| 微信小程序 | `0.6.0` | 无需重传 |
| `@ai-todo/agent-protocol` | `0.4.0` | 来源工具 + guidelines + `agent-tools.json` |

## 主要变化

- `AI_TODO_AGENT_TOOLS` 新增：`find`、带来源 `create`、按 `source` 列表、`update`/`done`/`delete`/`reschedule` by source。
- `AI_TODO_AGENT_GUIDELINES` 增补：幂等路径、`source` vs `x-client-source`、`source_meta` 语义、`--idempotency-key`。
- `skills/ai-todo/SKILL.md`：邮件 / 工单来源示例。
- `docs/agent-usage.md`：「外部来源与幂等」章节。
- `docs/api-design.md`：来源字段与 `lookup` 说明。
- 构建产物 `packages/agent-protocol/dist/agent-tools.json`。

## 发布说明

- 本批**不要求** CD 或微信小程序上传（组件 L1 与 v0.6.0 相同）。
- 验收：`pnpm --filter @ai-todo/agent-protocol build`、`pnpm test:api`。

详见 [v0.6.1-plan.md](./v0.6.1-plan.md)。
