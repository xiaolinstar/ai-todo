## 交付能力

### Tag（用户级词表）

- 表：`tags`、`reminder_tags`
- 创建/更新提醒时传 `tagNames[]`，服务端 upsert 并整组替换
- `GET /v1/tags?q=` 供小程序联想

### 搜索

- `GET /v1/reminders?q=` — 匹配 title、notes、tag 名、跟踪 text
- `GET /v1/reminders?tag=` — 精确 tag 筛选
- 新增 `sort=updated_at`

### 事项跟踪

- 表：`reminder_track_entries`
- `POST /v1/reminders/{id}/track-entries` — append-only，服务端生成 `dateLabel`（用户时区 MM-DD）
- 与 `notes` 并存

### CLI

- `--tag`（create/update）、`--q` / `--tag`（list）
- `reminder track add <id> <text>`

### 小程序

- 列表页搜索栏 + tag chip 筛选
- 创建/编辑页 tag 输入
- 编辑页备注下跟踪列表与即时追加
