Official ai-todo API service icon assets live here.

- `ai-todo.svg`: default service icon used by the API landing page and favicon routes.
- `ai-todo-light.svg` / `ai-todo-light.png`: 1024 x 1024 light-theme service icon.
- `ai-todo-dark.svg` / `ai-todo-dark.png`: 1024 x 1024 dark-theme service icon.
- `ai-todo-favicon-light.svg` / `ai-todo-favicon-light.png`: 512 x 512 light favicon variant.
- `ai-todo-favicon-dark.svg` / `ai-todo-favicon-dark.png`: 512 x 512 dark favicon variant.
- `ai-todo-adaptive.svg`: currentColor adaptive mark for contexts that supply their own background.
- `gonganlianwang.png`: public security registration icon used in the footer.
